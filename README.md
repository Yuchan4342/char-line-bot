## Desctiption / おチャー bot とは?
LINE bot replying message + "チャー".  
You can append favorite string by 「文字列切替」 menu below.  

このLINE botはあなたのメッセージの後ろに"チャー"をつけて返します。
トーク画面下の「文字列切替」メニューから後ろにつける文字に好きな文字を設定できます。
